OVERVIEW

This is a mini space game built into the webpage itself. While following the LinkedIn Learning course "Builing the Web", I made a handful of small projects to practice and strength my web development skills. The last of which was this space game. Feel free to inspect the webpage and view the code, their is very little HTML and CSS this project was mostly built with JavaScript.

Wanting to obtain a greater understanding of JavaScript I took on the LinkedIn Learning "JavaScript Essential Training" course that gave me greater insight into web development as a whole and more of JavaScripts quirks and oddities.

HOW TO PLAY
  • Use the keys A & D to move.
  • Use the SPACE key to fire a bullet (bullet fires from wherever the your space ship is positioned).
  • Refresh the page to replay the game.
