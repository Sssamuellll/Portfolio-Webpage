ANNIHILATION

GRADE: 84%

Platform | Windows
Engine | None, C++
Project Duration | April 2023 - May 2023 (2 Weeks)
Tools Used | Visual Studio, SDL2, Adobe Photoshop
Methodologies | Object Oriented Programming
OVERVIEW

Annihilation is a 2D Arcade Shooter University Project built purely out of C++ to practice Object Oriented Programming and inheritance while learning to utilise libraries to implement complex features such as text element, music and sound effects that enhance the game.
