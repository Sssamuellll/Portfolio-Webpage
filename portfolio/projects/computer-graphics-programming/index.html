OVERVIEW

A base demo application was provided before exploring PBR theory and researching various BRDF equations that could be implemented into this program. Along with the existing Blinn-Phong shader, I implemented two additional BRDF shaders. The purpose of the application is to demonstrate a comparison between Blinn-Phong's lighting solution and the modern PBR solutions as well as to practice computer graphcis programming. IBL was attempted by implementing a cubemap skybox within the established scene to eventually influence the objects lighting further.
